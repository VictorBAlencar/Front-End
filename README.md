# Estudo de Front-End

Este repositório foi criado para organizar meus estudos e exercícios práticos sobre desenvolvimento web front-end, focando no domínio de HTML, CSS e JavaScript.

## Estrutura de Aprendizado e Organização

O projeto está estruturado de forma cronológica e temática para facilitar a navegação e a revisão de conceitos. A separação por pastas segue a lógica de modularização por assunto:

### 1. HTML (HyperText Markup Language)

As pastas nesta seção são numeradas de forma sequencial (ex: `00_`, `01_`) para refletir o progresso nos conceitos fundamentais da linguagem. Cada diretório agrupa exercícios relacionados a uma categoria específica, tais como:

* Estrutura básica, metadados e declaração de documentos.
* Organização de conteúdo, cabeçalhos e parágrafos.
* Formatação de texto, semântica e estilos inline.
* Integração de mídia e navegação por links.

### 2. CSS e JavaScript (Futuras implementações)

Seguindo o mesmo padrão organizacional do HTML, os novos módulos serão categorizados por tópicos de complexidade crescente, mantendo a separação entre a estilização visual e a lógica de comportamento dinâmico.

## Tecnologias Utilizadas

* HTML5
* VS Code
* Git & GitHub

## Como visualizar

Para visualizar qualquer exercício localmente, basta navegar até a pasta desejada e abrir o arquivo `.html` em um navegador moderno.
