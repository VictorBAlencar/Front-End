# Estudo de Front-End

Este repositório foi criado para organizar meus estudos e exercícios práticos sobre desenvolvimento web front-end. O foco inicial é o domínio das tecnologias fundamentais: HTML, CSS e JavaScript.

## Estrutura do Projeto

Atualmente, o projeto conta com as seguintes seções:

### 1. Basics HTML
Contém os fundamentos da linguagem de marcação.
* **Sintaxe Básica (`syntax.html`)**: Demonstração da estrutura de um documento HTML5, incluindo:
    * Cabeçalhos (H1 a H6).
    * Parágrafos e quebras de linha.
    * Atributos de elementos (title, style).
    * Inserção de links e imagens com texto alternativo.

## Tecnologias Utilizadas
* HTML5
* VS Code
* Git & GitHub

## Como visualizar
Para visualizar os arquivos localmente, basta abrir o arquivo `.html` desejado em qualquer navegador moderno.
